# 📊 Student Performance Analysis

This project analyzes student academic performance using Python and data analysis libraries.

---

## 🚀 Features
- Calculate total and average marks
- Identify top-performing students
- Analyze performance trends
- Clean and process dataset

---

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

---

## 📂 Dataset
- students.csv containing marks (G1, G2, G3)

---

## ▶️ How to Run
1. Clone the repository
2. Install dependencies:
   pip install pandas matplotlib seaborn
3. Run the script:
   python analysis.py

---

## 📌 Output
- Total and average marks calculated
- Ready for visualization and further analysis
